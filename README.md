# WebClip

Webクリッピングを行います。Webページに対して本文抽出をHTMLタグ付きで行います。
![概要](./fig/abstract.png)

## Usage

```sh
ruby web_clip.rb [URL]

# e.g.
ruby web_clip.rb https://github.com/KenshoFujisaki/WebClip
```
